# 01 Introduction to Computers, Internet & Java

In this chapter, we learn about the basic components of a computer, the role of the internet, and the Java programming language.

## Simple Explanation
Java is a powerful, high-level programming language that is designed to be platform-independent. This means "Write Once, Run Anywhere."

## Key Points
-   **Hardware:** The physical components of a computer (CPU, RAM, Hard Disk).
-   **Software:** The programs that run on the hardware.
-   **JVM (Java Virtual Machine):** The engine that executes Java code.
-   **JDK (Java Development Kit):** The tools used to develop Java programs.

## Syntax Breakdown
```java
public class Main { // Class declaration
    public static void main(String[] args) { // Entry point of the program
        System.out.println("Hello World!"); // Output statement
    }
}
```

## Example 1: Hello World
This is the most basic Java program that prints a message to the console.

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Welcome to Java Programming!");
    }
}
```

## Example 2: Simple Output
You can use `print` and `println` to display text.

```java
public class OutputDemo {
    public static void main(String[] args) {
        System.out.print("This is ");
        System.out.println("on the same line.");
    }
}
```
