# Java OOP Prep: Beginner to Intermediate

Welcome to the **Java OOP Prep** repository! This repository is designed as a complete teaching resource for absolute beginners learning Java, based on the curriculum of **"Java How to Program (10th Edition)"**.

## 🎯 Course Overview
This course covers the fundamentals of Java programming, from basic input/output to object-oriented programming concepts like classes and arrays. Each chapter is structured to provide a simple explanation, practical code examples, and exercise sets to reinforce learning.

## 🚀 How to Run Java Programs
To run the examples in this repository, follow these steps:

1.  **Install JDK:** Download and install the Java Development Kit (JDK 8 or later).
2.  **Choose an IDE:** You can use a simple text editor (like VS Code) or an IDE (like IntelliJ IDEA, Eclipse, or NetBeans).
3.  **Compile the file:**
    ```bash
    javac FileName.java
    ```
4.  **Run the program:**
    ```bash
    java FileName
    ```
